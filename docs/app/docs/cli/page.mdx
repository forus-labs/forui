# CLI

Forui includes a CLI that generates color schemes, typographies and styles in your project. These styles can be directly
modified to fit your design needs. No additional installation is required. Neither does the CLI increase your
application's bundle size.

## init

Use the `init` command to create a `forui.yaml` configuration file in your project's directory.

```shell copy
dart run forui init
```

```shell copy
Usage: forui init
-h, --help     Print this usage information.
-f, --force    Overwrite existing files if they exist.
```

## color

### create

Use the `color create` command to generate color scheme(s) in your project. The color schemes are generated in
`lib/theme/color.dart` by default.

```shell copy
dart run forui create [color schemes]
```

```shell copy
Usage: forui color create [color schemes]
-h, --help      Print this usage information.
-a, --all       Create all color schemes.
-f, --force     Overwrite existing files if they exist.
-o, --output    The output directory or file, relative to the project directory.
                (defaults to "lib/theme/color.dart")
```

### ls

Use the `color ls` command to list all available color schemes.

```shell copy
dart run forui ls
```

```shell copy
Usage: forui color ls
-h, --help    Print this usage information.
```
