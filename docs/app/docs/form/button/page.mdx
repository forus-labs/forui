## Usage

### `FButton(...)`

```dart copy
FButton(
  onPress: () {},
  child: const Text('Button'),
);
```

### `FButton.raw(...)`

```dart copy
FButton.raw(
  onPress: () {},
  child: const Text('Button'),
);
```
