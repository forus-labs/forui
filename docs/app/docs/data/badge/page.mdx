# Badge
A badge draws attention to specific information, such as labels and counts. Use badges to display status, notifications, or small pieces of information that need to stand out.

## Usage

### Basic Usage
Create a badge with just a label.

```dart copy
FBadge(label: const Text('New'));
```

### With Icon
Add an icon to enhance the badge's visual appeal.

```dart copy
FBadge(
  label: const Text('New'),
  icon: const Icon(FIcons.bell, size: 12),
);
```

### Raw Constructor
Use the raw constructor for complete customization of the badge content.

```dart copy
FBadge.raw(
  builder: (context, style) => Row(
    mainAxisSize: MainAxisSize.min,
    children: [
      Icon(FIcons.star, size: 12, color: style.color),
      const SizedBox(width: 4),
      Text('Premium', style: style.textStyle),
    ],
  ),
);
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `Widget` | - | The main content of the badge |
| `icon` | `Widget?` | `null` | An optional icon to display alongside the label |
| `style` | `FBadgeStyle` | `FBadgeStyle.primary` | The visual style of the badge |
| `builder` | `Widget Function(BuildContext, FBadgeStyle)` | - | A builder function for the raw constructor |

## Best Practices

1. Use badges sparingly to avoid visual clutter
2. Keep badge text short and concise
3. Choose appropriate styles based on the context:
   - Primary: Important information
   - Secondary: Less important information
4. Use icons to enhance visual communication
5. When using the raw constructor, ensure custom styling matches the overall design system
