# Tile

A tile that is typically used to group related information together.

    <LinkBadge label="API Reference" href="https://pub.dev/documentation/forui/latest/forui.widgets.tile/"/>
</LinkBadgeGroup>

<Callout type="info">

</Callout>

## Usage

### `FTile(...)`

```dart copy
FTile(
  prefixIcon: Icon(FIcons.globe),
  title: const Text('Title'),
  subtitle: const Text('subtitle'),
  details: const Text('details'),
  suffixIcon: Icon(FIcons.chevronRight),
  semanticsLabel: 'Label',
  enabled: true,
  onPress: () {},
  onLongPress: () {},
);
```
