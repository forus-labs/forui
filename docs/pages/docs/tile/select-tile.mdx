# Select Tile
A select tile represents a single selection.

<LinkBadgeGroup>
    <LinkBadge label="API Reference" href="https://pub.dev/documentation/forui/latest/forui.widgets.tile/forui.widgets.select-tile-group-library.html"/>
</LinkBadgeGroup>

</Callout>

## Usage

### `FSelectTile(...)`

```dart
FSelectTile(
  title: const Text('Title'),
  subtitle: const Text('subtitle'),
  details: const Text('details'),
  suffixIcon: FIcon(FAssets.icons.list),
  semanticLabel: 'Label',
  enabled: true,
  value: 1,
);
```

### `FSelectTile.suffix(...)`

```dart
FSelectTile.suffixed(
  prefixIcon: FIcon(FAssets.icons.chevronRight),
  title: const Text('Title'),
  subtitle: const Text('subtitle'),
  details: const Text('details'),
  semanticLabel: 'Label',
  enabled: true,
  value: 1,
);
```
