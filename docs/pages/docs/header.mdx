# Header
A header contains the page's title and navigation actions.
It is typically used on pages at the root of the navigation stack.

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab>
    <Widget name='header'/>
  </Tabs.Tab>
  <Tabs.Tab>
        ```dart
        FHeader(
          title: 'Edit Alarm',
          actions: [
            FHeaderAction(
              icon: FAssets.icons.alarmClock,
              onPress: () {},
            ),
            FHeaderAction(
              icon: FAssets.icons.plus,
              onPress: () {},
            ),
          ],
        )
        ```
  </Tabs.Tab>
</Tabs>

## Usage

### `FHeader(...)`

```dart
FHeader(
  title: 'Edit Alarm',
  actions: [
    FHeaderAction(
      icon: FAssets.icons.alarmClock,
      onPress: () {},
    ),
    FHeaderAction(
      icon: FAssets.icons.plus,
      onPress: () {},
    ),
  ],
)
```
