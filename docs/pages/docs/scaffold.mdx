## Usage

### `FScaffold(...)`

```dart
FScaffold(
  header: FHeader(
    title: const Text('Settings'),
    actions: [
      FHeaderAction(
        icon: FAssets.icons.ellipsis,
        onPress: () {},
      ),
    ],
  ),
  contentPad: false,
  content: Column(
    children: [
      FCard(
        title: const Text('Notification'),
        subtitle: const Text('You have 3 unread messages.'),
        child: FButton(label: const Text('Read messages'), onPress: () {}),
      ),
    ],
  ),
  footer: const Text('Created by Forus Labs'),
);
```
