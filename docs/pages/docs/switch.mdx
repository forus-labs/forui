# Switch
A switch that allows the user to toggle between checked and unchecked. It can also be used in a form.

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab>
    <Widget name='switch' query={{enabled: 'true'}}/>
  </Tabs.Tab>
  <Tabs.Tab>
    ```dart
    FSwitch();
    ```
  </Tabs.Tab>
</Tabs>

## Usage

### `FSwitch(...)`

```dart
FSwitch(
  enabled: true,
  initialValue: true,
  autofocus: true,
  onChanged: (value) {},
);
```
