## Usage

### `FSwitch(...)`

```dart
FSwitch(
  label: const Text('Airplane Mode'),
  description: const Text('Turn on airplane mode to disable all wireless connections.'),
  error: const Text('Please turn on airplane mode.'),
  semanticLabel: 'Airplane Mode',
  value: true,
  onChanged: (value) {},
  enabled: true,
  autofocus: true,
);
```
