# Switch
A switch that allows the user to toggle between checked and unchecked. It can also be used in a form.

    <LinkBadge label="API Reference" href="https://pub.dev/documentation/forui/latest/forui.widgets.switch_/forui.widgets.switch_-library.html"/>
</LinkBadgeGroup>

## Usage

### `FSwitch(...)`

```dart
FSwitch(
  label: const Text('Airplane Mode'),
  semanticLabel: 'Airplane Mode',
  enabled: true,
  initialValue: true,
  autofocus: true,
  onChanged: (value) {},
);
```
