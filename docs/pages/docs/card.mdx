# Card
Displays a card typically with a title, subtitle, and child widget.

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab>
      <Widget name='card' query={{}}/>
  </Tabs.Tab>
  <Tabs.Tab>
      ```dart
      FCard(
          title: 'Notification',
          subtitle: 'You have 3 unread messages.',
      );
      ```
  </Tabs.Tab>
</Tabs>

## Usage

### `FCard(...)`

```dart
FCard(
  title: 'Notification',
  subtitle: 'You have 3 unread messages.',
  child: const FButton('Read messages', onPress: () {}),
);
```

### `FCard.raw(...)`

```dart
FCard.raw(
  child: const Placeholder(),
);
```
