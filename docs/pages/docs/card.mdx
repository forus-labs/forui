## Usage

### `FCard(...)`

```dart
FCard(
  title: const Text('Notification'),
  subtitle: const Text('You have 3 unread messages.'),
  child: FButton(label: const Text('Read messages'), onPress: () {}),
);
```

### `FCard.raw(...)`

```dart
FCard.raw(
  child: const Placeholder(),
);
```
