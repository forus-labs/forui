# Checkbox
A control that allows the user to toggle between checked and not checked. It can also be used in a form.

For touch devices, a [switch](/docs/switch) is generally recommended over a checkbox.

## Usage

### `FCheckbox(...)`

```dart
FCheckbox(
  enabled: true,
  initialValue: true,
  autofocus: true,
  onChanged: (value) {},
);
```
