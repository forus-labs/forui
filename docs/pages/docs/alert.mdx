## Usage

### `FAlert(...)`

```dart
FAlert(
  icon: FAlertIcon(icon: FAssets.icons.badgeAlert),
  title: const Text('Heads Up!'),
  subtitle: const Text('You can add components to your app using the cli.'),
);
```
