# Progress
Displays an indicator showing the completion progress of a task, typically displayed as a progress bar.

## Usage

### `FProgress(...)`

```dart copy
FProgress(
  style: FLinearProgressStyle(...),
  semanticsLabel: 'Label',
  value: 0.6,
  duration: const Duration(milliseconds: 500),
);
```

### `FProgress.circularIcon(...)`

```dart copy
FProgress.circularIcon(
  style: FCircularIconProgressStyle(...),
  semanticsLabel: 'Label',
  duration: const Duration(milliseconds: 500),
);
```

## Examples

### Determinate Linear Progress

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab>
    <Widget name='progress' variant='linear'/>
  </Tabs.Tab>
  <Tabs.Tab>
    ```dart copy
    class LinearProgress extends StatefulWidget {
      @override
      State<LinearProgress> createState() => _State();
    }

    class _State extends State<LinearProgress> {
      late Timer timer;
      double value = 0.2;

      @override
      void initState() {
        super.initState();
        timer = Timer(const Duration(milliseconds: 800), () => setState(() => value = 0.7));
      }

      @override
      Widget build(BuildContext context) => Column(
       children: [
         FProgress(value: value),
         FButton(label: const Text('Reset'), onPress: () => setState(() {
           value = 0.2;
           timer.cancel();
           timer = Timer(const Duration(milliseconds: 800), () => setState(() => value = 0.7));
         })),
       ],
     );

      @override
      void dispose() {
        timer.cancel();
        super.dispose();
      }
    }
    ```
