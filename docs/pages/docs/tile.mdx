# Tile
A tile that is typically used to group related information together.

<LinkBadgeGroup>
    <LinkBadge label="API Reference" href="https://pub.dev/documentation/forui/latest/forui.widgets.tile/forui.widgets.tile-library.html"/>
</LinkBadgeGroup>

Tiles can be grouped together inside a [tile group](/docs/tile-group).

## Usage

### `FTile(...)`

```dart
FTile(
  prefixIcon: FIcon(FAssets.icons.globe),
  title: const Text('Title'),
  subtitle: const Text('subtitle'),
  details: const Text('details'),
  suffixIcon: FIcon(FAssets.icons.chevronRight),
  semanticLabel: 'Label',
  enabled: true,
  onPress: () {},
  onLongPress: () {},
);
```
