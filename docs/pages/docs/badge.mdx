# Badge
A badge typically draws attention to specific information, such as labels and counts.

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab>
      <Widget name='badge' query={{}}/>
  </Tabs.Tab>
  <Tabs.Tab>
      ```dart
      FBadge(label: 'Badge');
      ```
  </Tabs.Tab>
</Tabs>

## Usage

### `FBadge(...)`

```dart
FBadge(label: 'Badge');
```

### `FBadge.raw(...)`

```dart
FBadge.raw(builder: (context, style) => Text('Badge'));
```
