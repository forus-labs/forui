# Nested Header
A nested header contains the page's title and actions (including pop navigation).
It is typically used on pages that are not at the root of the navigation stack.

<Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
        <Widget name='nested-header'/>
    </Tabs.Tab>
    <Tabs.Tab>
        ```dart
        FNestedHeader(
          title: 'Appointment',
          leftActions: [
            FNestedHeaderAction.back(onPress: () {}),
          ],
          rightActions: [
            FNestedHeaderAction(
              icon: FAssets.icons.info,
              onPress: () {},
            ),
            FNestedHeaderAction(
              icon: FAssets.icons.plus,
              onPress: () {},
            ),
          ],
        );
        ```
    </Tabs.Tab>
</Tabs>

## Usage

### `FNestedHeader(...)`

```dart
FNestedHeader(
  title: 'Appointment',
  leftActions: [
    FNestedHeaderAction.back(onPress: () {}),
  ],
  rightActions: [
    FNestedHeaderAction(
      icon: FAssets.icons.info,
      onPress: null,
    ),
    FNestedHeaderAction.x(onPress: () {}),
  ],
);
```
