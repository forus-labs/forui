## Usage

### `FRadio(...)`

```dart
FRadio(
  label: const Text('Default'),
  description: const Text('The description of the default option.'),
  error: const Text('Please select the default option.'),
  semanticLabel: 'Default',
  value: true,
  onChanged: (value) {},
  enabled: true,
  autofocus: true,
);
```
