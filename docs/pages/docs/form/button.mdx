## Usage

### `FButton(...)`

```dart copy
FButton(
  label: const Text('Button'),
  onPress: () {},
);
```

### `FButton.raw(...)`

```dart copy
FButton.raw(
  child: const Text('Button'),
  onPress: () {},
);
```
