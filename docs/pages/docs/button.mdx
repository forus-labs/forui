# Button
A button.

## Usage

### `FButton(...)`

```dart
FButton(
  label: const Text('Button'),
  onPress: () {},
);
```

### `FButton.raw(...)`

```dart
FButton.raw(
  child: const Text('Button'),
  onPress: () {},
);
```
