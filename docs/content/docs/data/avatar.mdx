## CLI

To generate and customize this style:

```shell copy
dart run forui style create avatar
```

## Usage

### `FAvatar(...)`

```dart copy
FAvatar(
  image: const NetworkImage('https://example.com/profile.jpg'),
  style: (style) => style.copyWith(...),
  size: 40.0,
  semanticsLabel: 'User avatar',
  fallback: const Text('JD'),
);
```

### `FAvatar.raw(...)`

```dart copy
FAvatar.raw(
  child: const Text('MN'),
  style: (style) => style.copyWith(...),
  size: 40.0,
);
```
