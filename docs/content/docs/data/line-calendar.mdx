## CLI

To generate and customize this style:

```shell copy
dart run forui style create line-calendar
```

## Usage

### `FLineCalendar(...)`

```dart copy
FLineCalendar(
  control: FLineCalendarControl.managed(
    controller: FCalendarController.date(...),
    initial: DateTime.now(),
    toggleable: false,
    selectable: (date) => true,
    onChange: (date) {},
  ),
  style: (style) => style.copyWith(...),
  initialScrollAlignment: AlignmentDirectional.center,
  physics: const AlwaysScrollableScrollPhysics(),
  cacheExtent: 100,
  keyboardDismissBehavior: ScrollViewKeyboardDismissBehavior.manual,
  builder: (context, data, child) => child!,
  start: DateTime(1900),
  end: DateTime(2050),
  initialScroll: DateTime.now(),
  today: DateTime.now(),
);
```
