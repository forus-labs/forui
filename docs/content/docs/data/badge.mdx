## CLI

To generate and customize this style:

```shell copy
dart run forui style create badges
```

## Usage

### `FBadge(...)`

```dart copy
FBadge(
  child: const Text('Badge'),
  style: FBadgeStyle.primary(),
);
```

### `FBadge.raw(...)`

```dart copy
FBadge.raw(
  builder: (context, style) => Text('Badge'),
  style: FBadgeStyle.primary(),
);
```
